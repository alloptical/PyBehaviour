## Prerequisites
* Python (3.4 recommended, if on Windows: [winpython](http://winpython.github.io))
  * PyQT5
* Arduino (MEGA 2560 recommended)
  * [TaskScheduler](http://playground.arduino.cc/Code/TaskScheduler)
